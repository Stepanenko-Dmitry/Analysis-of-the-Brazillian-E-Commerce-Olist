# Analysis-of-the-Brazillian-E-Commerce-Olist: Retention, Revenue & LTV

## О проекте

В данном проекте проведен продуктовый анализ e-commerce платформы с фокусом на:

- поведение пользователей  
- retention (удержание)  
- повторные покупки  
- выручку (revenue)  
- жизненную ценность пользователя (LTV)  

Цель — выявить причины низкого удержания пользователей и определить ключевые точки роста продукта.

---

## Задачи

- Проанализировать поведение пользователей  
- Оценить уровень retention  
- Исследовать структуру выручки  
- Рассчитать и интерпретировать LTV  
- Найти факторы, влияющие на возврат пользователей  
- Сформулировать продуктовые рекомендации  

---

## Данные

В проекте используются данные e-commerce платформы, включающие:

- пользователей (customers)  
- заказы (orders)  
- товары и категории (products)  
- отзывы (reviews)
- и т.д.  

---

## 🔍 Основные этапы анализа

1. **Exploratory Data Analysis (EDA)**  
   - структура данных  
   - базовые распределения  

2. **Анализ пользователей**  
   - количество заказов на пользователя  
   - доля одноразовых покупателей  

3. **Анализ retention**  
   - доля пользователей с повторными покупками  
   - поведенческие паттерны  

4. **Revenue анализ**  
   - вклад пользователей в выручку  
   - зависимость от новых пользователей  

5. **LTV анализ**  
   - оценка жизненной ценности пользователя  
   - распределение LTV  

6. **Анализ категорий**  
   - популярные категории  
   - связь с retention  

7. **Анализ оценок**  
   - уровень удовлетворенности  
   - влияние на возврат  

---

## Ключевые инсайты

- Большинство пользователей совершает только одну покупку  
- Retention находится на низком уровне  
- Повторные покупки происходят редко  
- Выручка формируется преимущественно за счет новых пользователей  
- LTV пользователей низкий  
- Популярные категории не обеспечивают возврат пользователей  
- Даже категории с потенциалом повторных покупок (например, health_beauty) не удерживают пользователей  
- Оценки товаров находятся на среднем уровне  

---

## Главный вывод

Проблема продукта — не привлечение пользователей, а их удержание.

Платформа эффективно конвертирует пользователей в первую покупку,  
но не создает условий для повторного взаимодействия.

---

## Бизнес-логика

Низкий retention —  
мало повторных покупок —  
низкий LTV —  
зависимость от привлечения —  
ограничение роста
